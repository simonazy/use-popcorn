# use-popcorn
#### Main Components structure
```jsx
<Main>
  <Nav>
    <Search />
  <Nav/>
  <Box><MovieList>/</Box>
  <Box>
      <>
        <WatchedSummary/>
        <WatcheMovieList/>
      </>
  </Box>
<Main/>
```
#### Other Components
-`<MovieDetails/>`

-`<StarRating />` from [MUI](https://mui.com/material-ui/react-rating/#sizes) 

-`<Loader /> `

-`<Error />`

`useEffect()` 

`useRef()` 

and a custom Hook `useKey()`
